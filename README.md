## This project was bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app).

### How to start？

1. install the modules

   ```
   $ npm install
   ```

2. starting up api ( build by json-sever )

   ```
   $ cd api/
   $ npm start
   ```

3. In the root directory , run

   ```
   $ npm start
   ```
